# Broxy (blooket but proxy)

</div>
broxy is a fan-made game site built in the style of blooket. it's basically blooket but hollowed out and turned into a game site. we also have a massive library of free browser games that you can play without signup, powered by LuminSDK. i got this idea from @whydoistilldothis on discord (tysm)

## features
**huge game library** — over 1,000 browser games using LuminSDK. (https://luminsdk.com/)

**blooket-inspired design** — inspired by blooket's UI design

**no sign up required** — just hit play and start gaming


>⚠️ broxy is a parody project and is not affiliated with, endorsed by, or related to Blooket LLC. in any way.
